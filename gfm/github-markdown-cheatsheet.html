Multi-Column Layout (HTML Table trick)
<table><tr> <td width="50%"> **Left column** Any markdown here. </td> <td width="50%"> **Right column** </td> </tr></table>
  • Blank line required before Markdown inside <td>
  • Supports code blocks, lists, images inside cells
  • Use align="center" on <td> for centered content
  • Works with 2, 3, or 4 columns
Left ColumnUse for before/after, pros/cons, input/output comparisons.
Right ColumnAny valid Markdown renders here including code blocks.
